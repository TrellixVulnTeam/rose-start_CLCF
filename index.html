Why RoseStart?

In conversations revolving around computer science, math and robotics, our opinions and ideas are often overlooked. When that happens, we tend to think that our ideas are not good enough to be heard. Most women and non-binary experience some sort of gender exclusion from the field of technology. As young women in tech ourselves, our experiences led us to take action to help each other and those who have to work twice as hard as men in tech.

There is a misconception that suggests men are smarter than women and non-binary people. To this day, we still struggle to be part of the field of technology due to the lack of resources provided to us. When we arenâ€™t fully granted the opportunity to explore our interests and capabilities, society then defines technology as a field directly and exclusively for men.

Our Mission Statement

With the goal of breaking the stigma that puts down women and non-binary in tech, we wanted to create a platform where women and non-binary can explore ways in which they can improve their skills and start building their career. This website is dedicated to making it easier for those who experience exclusion in tech to plan out their careers in these areas.

Things You Need To Know About
  • Sorting Algorithms

  • Bubble Sort: a sorting algorithm that repeatedly swaps the adjacent elements if they are in wrong order.

  • Quicksort: a Divide and Conquer algorithm. It picks an element as pivot and partitions the given array around the picked pivot.

  • Merge Sort: also a Divide and Conquer algorithm. t divides the input array into two halves, calls itself for the two halves, and then merges the two sorted halves.

  • Insertion Sort: The array is virtually split into a sorted and an unsorted part. Values from the unsorted part are picked and placed at the correct position in the sorted part.

  • Heap Sort: a comparison based sorting technique based on Binary Heap data structure.

  • Data Structures

  • Array: data structure consisting of a collection of elements, each identified by at least one array index or key

  • Linked List: linked list is a linear collection of data elements whose order is not given by their physical placement in memory

  • Stack/Queue: Stack is a linear data structure which follows a particular order in which the operations are performed. The order may be LIFO(Last In First Out) or FILO(First In Last Out).

  • Trees: a tree is a widely used abstract data type that simulates a hierarchical tree structure, with a root value and subtrees of children with a parent node, represented as a set of linked nodes.:

  • Heap: a heap is a specialized tree-based data structure which is essentially an almost complete tree that satisfies the heap property

  • Hash Table:a hash table is a data structure that implements an associative array abstract data type, a structure that can map keys to values

  • Graphs: A graph is data structure made up of two sets called Vertices and Edges which represent relations between two vertices
